3️⃣ Design de Agentes

Agora a parte mais importante.

Cada veterinário roda um agente especializado.

Estrutura do Agente
AgentVet

Componentes internos:

Agent
 ├── Prompt Engine
 ├── Tool Engine
 ├── Memory Engine
 ├── Context Builder
 └── Language Profile

+

+
Pet Context
+
Consulta atual
+
