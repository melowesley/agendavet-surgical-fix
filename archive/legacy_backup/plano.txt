
# Plano de Revisao e Melhoria Completa do AgendaVet

## Visao Geral do Projeto

O AgendaVet e um sistema veterinario composto por
- Web App (Vite + React) - Painel admin e portal do tutor
- Vet App (ExpoReact Native) - App mobile do veterinario (botoes em desenvolvimento, sem fluxos funcionais)
- Tutor App (ExpoReact Native) - App mobile do tutor
- Templete Web (Next.js) - Referencia futura (coexiste, nao sera priorizado)

```mermaid
graph TD
    subgraph Monorepo AgendaVet
        WEB[Web App (Vite + React)]
        VET[Vet App (Expo)]
        TUTOR[Tutor App (Expo)]
        NEXT[Templete Web (Next.js) - Referencia]
    end
    DB[Supabase (PostgreSQL + Auth + RLS)]
    WEB -- DB
    VET -- DB
    TUTOR -- DB
    NEXT -- DB
```
