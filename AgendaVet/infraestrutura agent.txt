gentVet AI Infrastructure
Clinical Copilot + Agent Control System for Veterinary Clinics

AgentVet é uma infraestrutura de inteligência artificial para clínicas veterinárias, integrada ao sistema AgendaVet.
O sistema oferece um copiloto clínico assistido por IA, além de um controle centralizado de agentes, permitindo que cada veterinário tenha sua própria IA especializada operando dentro do sistema.

A arquitetura foi projetada para permitir:

Assistência clínica em tempo real durante consultas

Automação de tarefas administrativas

Interação completa por chat com o sistema

Personalização da IA por veterinário

Controle remoto e atualização dinâmica de agentes

Visão Geral

O AgentVet foi projetado para atuar como um copiloto clínico inteligente, fornecendo suporte durante consultas veterinárias sem substituir o julgamento profissional.

Cada veterinário possui um agente de IA dedicado, configurado com:

contexto clínico

histórico dos pacientes

estilo de comunicação personalizado

ferramentas médicas

memória operacional

Além disso, existe um sistema central de controle de agentes, que permite atualizar e gerenciar todas as IAs remotamente.

Conceito de Agentes

Cada veterinário possui um agente de IA isolado.

Exemplo:

AgentVet Dr. Cleyton
AgentVet Dra. Mariana
AgentVet Dr. Felipe

Cada agente possui sua própria configuração:

prompt do sistema

estilo de linguagem

permissões de ferramentas

memória contextual

histórico de interações

Isso garante personalização e isolamento de dados por profissional.

Componentes Principais

Responsável por montar o contexto clínico do paciente.

Funções principais:

Sistema de Controle de Agentes

O sistema possui um controle central de agentes, chamado informalmente de AI Control Brain.

Esse sistema permite gerenciar todas as IAs do AgendaVet.

Exemplo de comandos administrativos:

/agents list
/agent update_prompt cleyton
/agent reload cleyton
/agent disable_tool cleyton dosage

Personalização de Linguagem

Cada veterinário pode configurar a forma como a IA se comunica.

Exemplo:

Tom formal
Tom amigável
Tom técnico

A IA pode também aprender o estilo do veterinário a partir das interações.

Exemplo de resposta personalizada:

"De acordo com os registros do sistema, a última vacina foi aplicada em 14/02/2026.

Precisa de ajuda com mais alguma coisa, Dr. Cleyton?"
Segurança e Isolamento de Dados

O sistema utiliza Row Level Security (RLS) no Supabase.

Isso garante que:

veterinários só acessam dados da própria clínica

dados de pacientes são isolados por login

interações com IA são auditáveis

Regra básica:

vet_id = auth.uid()
Versionamento de Prompts

Prompts do sistema são versionados.

Tabela:

prompt_versions

id
version
prompt_text
created_at

Isso permite:

atualizar prompts de todos os agentes

testar prompts em agentes específicos

rollback de versões

Exemplo:

deploy_prompt v12 → todos veterinários
deploy_prompt v12 → apenas Dr Cleyton
Logging e Auditoria

Isso permite:

Conclusão

O AgentVet não é apenas um chatbot.

Ele representa uma infraestrutura de inteligência artificial para clínicas veterinárias, combinando:

copiloto clínico

automação administrativa

agentes personalizados por veterinário

controle central de IA
